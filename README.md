# vevor_inverter_5kVa
Simple python implementation of a 5kVa Vevor (voltronic, ). Reads and writes basic values and published via MQTT to they can be read by e.g., Home Assistant. Written for readability, not efficiency. Should even run on raspberry pi zero (W)
